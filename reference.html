On this page you’ll find a quick reference to common HTML terms and a table of frequently used named character entities.

HTML Terms

Attribute
An HTML element property that provides additional information or functionality for an element. Attributes appear as a name/value pair with the value contained in quotation marks.
Block-Level Element
HTML tags that create blocks of content within the document such as headings and paragraphs. Block-level elements typically stack one on top of each other based on the order they appear in the code.
Child Element
Any HTML element that is contained inside another HTML element. Child element tags must close before their parent element tags.
Closing Tag
Appears after an element’s content and indicates the end of an element. It is identical to an element’s opening tag with the addition of a forward slash (/) prior to the tag’s character.
Comment
Reference information added to the code. May include an overview of more complex code, a reminder for future authors, or communications within a team.
CMS
Short for Content Management System. Refers to software designed to allow users to create and maintain website content. Examples include Joomla! and WordPress.
CSS
Cascading Style Sheets (CSS) is a presentation language used to control the presentation and layout of web pages.
Doctype
Short for Document Type Declaration. Appears at the top of HTML documents to trigger standards mode and identify the HTML version used.
DOM
The Document Object Model represents HTML documents as a tree of nodes. JavaScript uses the DOM to access or modify page content.
Elements
HTML elements consist of an HTML tag and the content inside it. The term is often used interchangeably with “tags,” though technically tags refer only to the markup.
FTP
File Transfer Protocol is a standard protocol used for transferring files between servers.
HTTP
Hypertext Transfer Protocol controls the formatting and transmission of HTML pages across web servers.
Inline-Level Elements
Elements that appear within the normal flow of text and do not start on a new line.
JavaScript
A scripting language used to control functionality and behavior on the web.
Metadata
Additional information about a document, typically stored in meta tags.
MIME Type
Multipurpose Internet Mail Extension types tell user agents what type of file or program is being transferred.
Parent Element
An HTML element that contains another element.
Replaced Element
An element whose content is replaced by external content such as images or videos.
Tag
Markup characters that define the type of content on a page. Most tags have both opening and closing forms.
URL
Uniform Resource Locator. The address of a document or resource on the web.
W3C
The World Wide Web Consortium, the standards body responsible for HTML specifications.
